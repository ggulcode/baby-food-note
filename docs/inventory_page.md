# P3. 창고 (인벤토리)

## 개요
보유한 식재료를 관리하고 큐브(소분된 재료)를 제작하는 페이지입니다.

## 기능 명세

### 1. 인벤토리 그리드
- **표시 조건**: 보유 수량이 1개 이상인 재료만 그리드에 노출.
- **UI 스타일**: 픽셀 아트 스타일의 재료 아이콘.
- **수량 표시**: 아이콘 하단에 `x{수량}` 배지(Badge) 표시.

### 2. 큐브 제작 (재료 입고)
- **프로세스**:
  1. '큐브 만들기' 또는 '재료 추가' 버튼 클릭.
  2. 전체 재료 목록(40종) 팝업 또는 드롭다운에서 재료 선택.
  3. **수량 입력**: 제작한 큐브의 개수(숫자) 입력.
  4. '확인' 클릭 시 인벤토리 수량 합산 (+).

## 데이터 구조
```typescript
interface Inventory {
  [ingredientId: string]: {
    count: number; // 큐브 개수
    lastUpdated: string;
  };
}
```
